Used to get the playlists for the supplied user.

Plugin Parameters:
	1) Username 		(the username to get the playlists from)
	2) Playlists 		(the number of playlists to get from the user above)
	3) Videos			(the number of videos per playlist to get from the user above)
	
Plugin Defaults:
	1) Username		:		''
	2) Playlists	:		10
	3) Videos		:		10
